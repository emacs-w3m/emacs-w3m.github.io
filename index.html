Emacs 上で w3m を動かそう

Emacs 上で動作するブラウザと言えば、普通 W3 のことですが、動作が非常に遅いので、なかなか常用しようという気になれません。

それに対して、w3m というテキストベースで動作するブラウザがあり、 非常に軽快に動作するので重宝しているのですが、 端末に移動しなければいけないのが面倒です。

そこで、w3m を HTML の rendering engine として使用し、表示とインタフェースのみを Emacs で動かすようにすればいいんじゃないだろうか、ということを考えました。

ダウンロード

  • 安定版 w3m_el-0.1.36.tar.gz
  • 開発版 w3m_el-0.2.17.tar.gz

w3m.elは、CVSを使って共同開発しているので、以下の手順で最新版が入手できます。 

% cvs -d :pserver:anonymous@cvs.namazu.org:/storage/cvsroot login
CVS password:   # Enter/Returnキーを叩く
% cvs -d :pserver:anonymous@cvs.namazu.org:/storage/cvsroot co emacs-w3m

また、ViewCVS でソースを閲覧することもできます。

インストール

何を置いても、w3m 本体が必要です。 本家のページなどから取得してインストールします。

次に、適当なディレクトリに w3m.el を置いて、以下の設定を ~/.emacs に追加してください。 

(autoload 'w3m "w3m" "Interface for w3m on Emacs." t)

後は、M-x w3m とすれば起動します。

メーリングリスト

w3m.el についての情報交換を行うためのメーリングリストを開設しています。 参加を希望される方は、emacs-w3m-ctl@namazu.org 宛に以下のような内容のメールを送ってください。 

subscribe ANATANO Namae

メーリングリスト本体のアドレスは、emacs-w3m@namazu.org です。 また、メーリングリストの過去記事も公開されています。
